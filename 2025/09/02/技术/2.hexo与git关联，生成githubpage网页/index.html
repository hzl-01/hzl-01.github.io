2.hexo与git关联，生成githubpage网页

正文内容

ok,这里默认有了git账号，并安装好了gitGit - Downloads
新建一个仓库 用户名.github.io 这里的用户名指的是你的gitHub用户名
打开cmd
输入npm install hexo-deployer-git –save 需要这个插件
进入hexo配置文件指定仓库路径就是那个_config.yml后缀的

然后输入hexo d，按提示输入你的账号密码
就可以了
打开 你的用户名.github.io 就可以了
下次更新文章之后`hexo clean && hexo g && hexo d 清理，生成，上传，单个执行
就可以看到网页更新了
