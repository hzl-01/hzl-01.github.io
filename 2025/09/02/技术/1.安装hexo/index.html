1.安装hexo

正文内容

安装hexo
需要nodeNode.js 中文网 — 下载 Node.js® https://nodejs.cn/en/download/prebuilt-installer
安装好，用node -v试试有没有安装成功
cmd里输入npm config set registry https://registry.npmmirror.com/
设置镜像源
去一个你想要的文件夹安装hexo的，上面资源栏输入cmd
hexo init blog #
cd blog
npm install 安装
hexo server 启动
下面是一些hexo的指令
hexo new “文章标题” 会出现在你文件夹的\source_posts下
依次执行下面的语句
hexo clean 清理
hexo g 生成
hexo s 启动
这样一来，你浏览器打开，localhost:4000 应该就可以了
