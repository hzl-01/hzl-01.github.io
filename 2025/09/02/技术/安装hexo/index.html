安装hexo
需要nodeNode.js 中文网 — 下载 Node.js®
安装好，用node -v试试有没有安装成功
cmd里输入npm config set registry https://registry.npmmirror.com/设置镜像源
去一个你想要的文件夹安装hexo的，上面资源栏输入cmd
hexo init blog # cd blog # 进入你的博客家园 npm install安装hexo server 启动下面是一些hexo的指令hexo new “文章标题” 会出现在你文件夹的\source_posts下 hexo clean && hexo g && hexo s` 清理，上传，并启动，可以单个执行
这样一来，你浏览器打开，localhost:4000 应该就可以了
