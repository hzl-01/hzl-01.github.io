Go与Python学习计划表

第1章 环境准备与入门 搭建完整的开发环境链。一切实践的基础。 必做。使用Go 1.22+和Python 3.12+。 2025-xx-xx P0 必须最先完成
第2章 计算机组成原理扫盲 理解二进制、编码等底层知识。巩固CS基础。 可快速浏览或跳过。 (可选)
第3章 基本数据类型、运算符 Go基础语法。 跳过。你已掌握。 (跳过)
第4章 Python新特性 Python的类型注解、海象运算符。 感兴趣可看，否则跳过。 (可选)
第5章 字符串操作 Go&Python字符串处理、编码问题。 快速浏览，关注中文字符处理部分。 2025-xx-xx
第6章 条件循环语句 Go控制流语法。 跳过。你已掌握。 (跳过)
第7章 复杂数据类型 Go切片底层原理、map使用。理解Go核心数据结构。 必看。重点是切片原理。 2025-xx-xx P1 重点
第8章 指针 Go指针、make/new区别、nil。理解Go内存模型。 快速回顾，确保概念清晰。 2025-xx-xx
第9章 函数 defer/panic/recover机制。 重点看 defer和错误处理部分。 2025-xx-xx P1 重点
第10章 结构体 Go结构体、方法、组合（模拟继承）。 快速浏览，巩固知识。 2025-xx-xx
第11章 接口 Go接口核心（鸭子类型、抽象）、错误处理。 快速回顾，确保理解透彻。 2025-xx-xx
第12章 包管理与编码规范 Go Modules实战、项目结构、规范。工程化起点。 必看。学习标准的Go项目管理和组织方式。 2025-xx-xx P1
第13章 并发编程 Go并发核心：channel, context。Go王牌特性。 重点看 channel和context的应用。 2025-xx-xx P0 核心
第14章 RPC基础 从零理解RPC概念、演进历史。理论铺垫。 快速观看，建立宏观认识。 2025-xx-xx
第15章 Go RPC体验 体验标准库RPC，改造过程。理解RPC的组成部分。 快速实践，了解底层机制。 2025-xx-xx
第16章 gRPC快速入门 Protobuf用法、gRPC四种流模式。核心通信技术。 精学。所有后续微服务的通信基础。 2025-xx-xx P0 核心
第17章 gRPC进阶 gRPC高级特性：metadata、拦截器、认证、验证、超时。生产级应用。 必精学。这是微服务稳定性的保障。 2025-xx-xx P0 核心难点
第18章 需求分析 电商系统、管理后台需求分析。了解业务全景。 阅读了解，知道要做什么东西。 2025-xx-xx
第19章 架构设计 单体到微服务的演进思想、拆分原则。架构思维。 精看。这是整个课程的设计总纲。 2025-xx-xx P0 思想纲领
第20章 YApi安装配置 API管理工具YApi的使用。前后端协作效率。 实践安装，后续开发会用到。 2025-xx-xx
第21章 Python ORM Peewee Python操作MySQL。底层数据服务的技术选型。 必看。这是Python微服务操作数据库的方式。 2025-xx-xx P1
第22章 Gin框架 Go Web框架Gin的核心用法：路由、中间件、参数绑定。HTTP网关层技术。 精学。这是对外提供HTTP服务的核心框架。 2025-xx-xx P1
第23章 用户服务-Service 第一个微服务实战：表设计、密码处理、grpc接口实现。微服务标准开发流程。 精学。这是后续所有服务的模板。 2025-xx-xx P0 项目模板
第24章 Web层-Gin基础架构 集成所有基础组件：Zap日志、Viper配置、服务发现、GRPC调用。Web层标准工程结构。 精学。学习如何组织一个标准的Go Web项目。 2025-xx-xx P0 项目模板
第25章 Web层-用户接口开发 具体业务开发：JWT认证、跨域、验证码、短信。核心业务功能实现。 实践。跟着做，巩固Gin和项目结构知识。 2025-xx-xx P1
第26章 服务注册发现 Consul核心概念、服务注册、健康检查。微服务基石一。 精学+实践。亲手搭建并集成Consul。 2025-xx-xx P0 服务治理
第27章 集成注册中心 将Service服务和Web层服务集成到Consul。 实践。跟着做，确保跑通。 2025-xx-xx P0 服务治理
第28章 负载均衡 负载均衡原理与算法、集成GRPC负载均衡。微服务基石二。 理解原理并实践。 2025-xx-xx P0 服务治理
第29章 配置中心 Nacos核心概念、集成到Go和Python服务。微服务基石三。 精学+实践。亲手搭建并集成Nacos。 2025-xx-xx P0 服务治理
第30章 商品服务-Service 商品业务CRUD开发。第二个微服务实战。 实践。模仿用户服务，巩固开发模式。 2025-xx-xx
第31章 商品服务-Web 商品相关HTTP接口开发。 实践。模仿用户Web层，巩固Gin开发。 2025-xx-xx
第32章 阿里云OSS服务 对象存储实战、前端直传、回调校验。文件服务解决方案。 必看。学习现代应用如何处理文件存储。 2025-xx-xx P1 通用能力
第33章 库存服务-Service 库存表设计、基础库存接口。分布式场景铺垫。 必看。为下一章的分布式锁做准备。 2025-xx-xx
第34章 分布式锁 超卖问题、分布式锁原理与实现（Redis锁及其陷阱）。面试绝对核心。 精学。本章是价值核心，必须彻底搞懂。 2025-xx-xx P0 核心难点
第35章 订单服务-Service 订单、购物车业务开发，调用多个服务。分布式事务铺垫。 必看。业务逻辑复杂，了解如何编排多个微服务。 2025-xx-xx P1
第36章 订单服务-Web 订单相关HTTP接口、集成支付宝支付。核心业务。 实践。支付功能是电商核心，务必搞懂流程。 2025-xx-xx P1
第37章 用户操作服务 收藏、地址等边缘业务。 快速开发，巩固模式。 2025-xx-xx
第38章 用户操作服务-Web 边缘业务HTTP接口。 快速开发，巩固模式。 2025-xx-xx
第39章 前后端联调 与Vue前端联调。全栈体验。 感兴趣可做，否则了解即可。 (可选)
第40章 分布式问题引入 分布式事务、幂等性理论引入。理论铺垫。 必看。建立对接下来两大难题的概念认知。 2025-xx-xx
第41章 分布式事务理论 CAP/BASE、2PC/TCC/最终一致性等方案详解。理论核心。 精学。不管代码如何变，这些理论是根本。 2025-xx-xx P0 理论核心
第42章 消息队列选型 MQ应用场景、技术选型（Kafka/RocketMQ/RabbitMQ）。技术选型思维。 了解为何选择RocketMQ。 2025-xx-xx
第43章 RocketMQ入门 RocketMQ核心概念、安装、Python操作。消息队列实战。 精学。学习消息队列的基本用法。 2025-xx-xx P1
第44章 可靠消息最终一致性 用RocketMQ解决分布式事务（库存归还）。项目最大难点。 绝对核心。这是最高频的分布式事务解决方案，必须吃透。 2025-xx-xx P0 核心难点
第45章 幂等性机制 幂等性原理与解决方案（令牌、防重表、唯一索引等）。生产环境必备。 绝对核心。必须掌握，否则线上会出大问题。 2025-xx-xx P0 核心难点
第46章 链路追踪引入 链路追踪概念、技术选型（Jaeger）。监控铺垫。 了解概念。 2025-xx-xx
第47章 Jeager与OpenTracing OpenTracing标准、Jaeger用法、Python集成。可观测性一。 精学。分布式系统排查问题的眼睛。 2025-xx-xx P1
第48章 Gin集成Jeager Go Web服务集成链路追踪。可观测性二。 实践。将Jeager集成到你的Gin项目中。 2025-xx-xx P1
第49章 熔断限流 Sentinel核心概念与用法（熔断、降级、限流）。服务稳定性基石。 精学。高可用微服务必须考虑的问题。 2025-xx-xx P1
第50章 API网关 API网关概念、功能、选型（Kong）。流量入口。 了解网关在架构中的角色。 2025-xx-xx
第51章 Kong实战 Kong基本功能：路由、负载、认证、集成Consul。网关实战。 实践。亲手用Kong管理你的API。 2025-xx-xx P1
第52章 Jenkins入门 CI/CD概念、Jenkins安装、流水线基础。自动化部署一。 了解自动化部署流程。 2025-xx-xx
第53章 Jenkins自动化部署 完整CI/CD流水线搭建，部署所有微服务。项目收官。 实践。完成部署会让你对项目有完整的掌控感。 2025-xx-xx P1 收官
