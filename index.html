hexo与git关联，生成githubpage网页

正文内容

ok,这里默认有了git账号，并安装好了gitGit - Downloads
新建一个仓库 用户名.github.io 这里的用户名指的是你的gitHub用户名
打开cmd
输入npm install hexo-deployer-git –save 需要这个插件
进入hexo配置文件指定仓库路径就是那个_config.yml后缀的

然后输入hexo d，按提示输入你的账号密码
就可以了
打开 你的用户名.github.io 就可以了
下次更新文章之后`hexo clean && hexo g && hexo d 清理，生成，上传，单个执行
就可以看到网页更新了

  • 本文核心观点：建立github page
  • 值得进一步探讨的问题：利用github action 进行cicd
  • 行动项：成功建立好github page

安装hexo

正文内容

安装hexo
需要nodeNode.js 中文网 — 下载 Node.js® https://nodejs.cn/en/download/prebuilt-installer
安装好，用node -v试试有没有安装成功
cmd里输入npm config set registry https://registry.npmmirror.com/
设置镜像源
去一个你想要的文件夹安装hexo的，上面资源栏输入cmd
hexo init blog #
cd blog
npm install 安装
hexo server 启动
下面是一些hexo的指令
hexo new “文章标题” 会出现在你文件夹的\source_posts下
依次执行下面的语句
hexo clean 清理
hexo g 生成
hexo s 启动
这样一来，你浏览器打开，localhost:4000 应该就可以了

  • 值得进一步探讨的问题：关联git仓库
